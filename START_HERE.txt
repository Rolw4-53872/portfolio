Change Default Language to Arabic:
→ Edit: assets/js/language.js
→ Find: currentLanguage: localStorage.getItem...
→ Change to: currentLanguage: 'ar'

→ Edit: assets/js/main.js

🔧 Troubleshooting
================================================

Images not showing?
→ Check image paths in index.html
→ Make sure files are in assets/images/
→ Use correct file names

Dark mode not working?
→ Check browser console (F12)
→ Make sure darkmode.js is loaded
→ Try in incognito window

Arabic text not displaying?
→ Make sure file is saved as UTF-8
→ Use a good text editor (VS Code)
→ Check translations.json
