⚡ الخطوات السريعة (5 دقائق)
================================================

1️⃣ افتح الموقع
   - اضغط ضغطة يمين على index.html
   - Open with → Browser
   - أو ببساطة اسحب الملف للمتصفح

2️⃣ غيّر معلوماتك الشخصية
   - اعدل index.html
   - غيّر الاسم والبريد والهاتف
   - غيّر روابط GitHub و LinkedIn

3️⃣ أضف صورك
   - ضع صورة الملف الشخصي في: assets/images/profile.png
   - ضع صورة الخلفية في: assets/images/banner.png
   - ضع صور المشاريع في: assets/images/projects/

4️⃣ أضف مشاريعك
   - افتح: assets/js/main.js
   - ابحث عن: projectsRenderer.projects
   - غيّر المشاريع بمعلوماتك

5️⃣ شارك الموقع!
   - حمّل المجلد على GitHub
   - أو على Netlify
   - أو أي خادم ويب

================================================
🎨 التخصيص البسيط
================================================

تغيير الألوان:
→ افتح: assets/css/style.css
→ ابحث عن: :root {
→ غيّر الألوان

تغيير اللغة الافتراضية:
→ افتح: assets/js/language.js
→ ابحث عن: currentLanguage
→ غيّر 'en' إلى 'ar'

🚀 النشر على الإنترنت
================================================

الخيار 1: GitHub Pages (مجاني)
- إنشاء حساب GitHub
- رفع المجلد كـ repository
- تفعيل Pages من الإعدادات

الخيار 2: Netlify (مجاني وسهل جداً)
- روح https://netlify.com
- اسحب المجلد هناك
- موقعك مباشرة يشتغل!

الخيار 3: Vercel (مجاني)
- روح https://vercel.com
- ربط حسابك GitHub
- نشر آلي عند كل تحديث
